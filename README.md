# Weather & Air Quality App

A simple weather and air quality application using Open-Meteo API.

## Features
- Current weather information
- Air quality index (AQI, PM2.5, PM10, NO2)
- Level & health advice
- Instant tab switching without page reload

## Tech Stack
- HTML, CSS, JavaScript
- Open-Meteo API, Maptiler API

## Future Improvements
- 7-day forecast
- Language switch (VN/EN)
- Store last searched city
